What is Scratch Programming?

Scratch is free block-based programming language designed especially for kids. It‚Äôs an open source platform developed by MiT(Massachusetts Institute of Technology) that gives kids a kickstart to learn how to code easily and at young age.


Scratch programming is a visual programming language in which you have to drag and drop blocks of code and attach them in a way you connect a jigsaw puzzle. It can be used online or offline on your system.
